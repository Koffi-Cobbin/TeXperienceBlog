Creating an Alert

  • To create an alert, visit any of our supported stores. Browse the store to view any item you're interested in.
  • After visiting the item page, (i.e the page that describes the particular item) copy the URL of the page.
  • Paste this URL in the URL section of the Create Alert Page
  • Now, create your alert.

    • We'll do the rest!

NOTE!

An incorrect URL will fail to create an alert!
